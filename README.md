## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Build for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.

## Sections

- **Hero Section**: Eye-catching introduction with business name and stats
- **Business Details**: Services offered and brands available
- **Contact Information**: Phone, email, address, and social media links
- **Privacy Policy**: Comprehensive privacy policy information

## Technologies Used

- React 18
- CSS3 with animations
- Google Fonts (Poppins)
- Intersection Observer API for scroll animations
